This repo hosts my portfolio website for GitHub Pages.

## How to use
1. Replace `YOUR_GITHUB_USERNAME` in `assets/js/main.js`.
2. Put your photo at `assets/images/profile.jpg`.
3. Add `resume.pdf` if you want a downloadable resume.
4. Commit and push to a repo named `yourusername.github.io`.

## Deployment
This site uses plain HTML/CSS/JS and will be served by GitHub Pages automatically for `username.github.io`.
